# Proyecto Final – Desarrollo Móvil II

## Descripción
Este proyecto corresponde al trabajo final del curso **Desarrollo Móvil II**, en el cual se aplican los conocimientos adquiridos sobre desarrollo de aplicaciones móviles modernas consite en la creeación de un juego basado en la serie **El Plan del Diablo**.

El objetivo principal es desarrollar una aplicación funcional, intuitiva y eficiente que nos permitar divertirnos en el juego, implementando buenas prácticas de desarrollo.
